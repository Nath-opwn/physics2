API功能概览

模拟管理

  • 创建新的流体模拟
  • 设置模拟参数和边界条件
  • 控制模拟运行（开始/暂停/停止）
  • 获取模拟状态和进度
  • 添加力和障碍物

数据分析

  • 获取流体场数据（速度、压力、涡量）
  • 计算统计数据和特征
  • 时间序列分析

可视化

  • 生成2D/3D可视化数据
  • 流线和粒子追踪
  • 等值面和等值线
  • 矢量场和标量场可视化

系统管理

  • 用户认证和授权
  • 系统状态监控
  • 资源使用情况
  • 错误日志和诊断

快速开始

认证

所有API请求都需要认证。使用以下方式获取访问令牌：

POST /api/users/token
Content-Type: application/x-www-form-urlencoded

username=your_username&password=your_password

创建模拟

创建一个新的流体模拟：

POST /api/simulation/initialize
Content-Type: application/json
Authorization: Bearer your_access_token

{
  "name": "涡环碰撞",
  "width": 100,
  "height": 100,
  "depth": 100,
  "viscosity": 0.1,
  "density": 1.0,
  "boundary_condition": "periodic"
}

控制模拟

开始运行模拟：

POST /api/simulation/control
Content-Type: application/json
Authorization: Bearer your_access_token

{
  "simulation_id": "sim_12345",
  "action": "start"
}

获取数据

获取模拟数据：

GET /api/simulation/data?simulation_id=sim_12345&field=velocity
Authorization: Bearer your_access_token
